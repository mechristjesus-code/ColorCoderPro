# Design Parameters — 144,000 Color Project

- Theme: Dark cosmic / luxury color intelligence
- Background: Deep space black #050508, subtle radial gradient overlays
- Primary accent: Gold #C9A84C / #E8C96A — luminous warmth
- Secondary accent: Soft white #F0EDE8 for text
- Muted text: #7A7A8C
- Card surfaces: rgba(255,255,255,0.04) with 1px border rgba(255,255,255,0.08)
- Font display: "Playfair Display" — elegant, authoritative
- Font body: "Inter" — clean readability
- Border radius: 8px cards, 4px small elements
- Glow effects: box-shadow with color-matched rgba for swatches
- Animations: subtle fade-in on scroll, color pulse on hover
- Color swatches: rounded squares with metallic/glossy/pearlescent CSS filters

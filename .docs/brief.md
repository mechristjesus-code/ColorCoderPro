# The 144,000 Color Project Platform
- One-line positioning: A living color intelligence platform assigning permanent identities to 1.87M color variants, discovering emotional meaning through human interaction and AI.
- Target users: Color researchers, designers, artists, educators, general public
- Core features:
  1. Landing page — vision, hierarchy, animated color universe, stats
  2. Color Explorer — browse 13 groups × 144 sub-classes × 1,000 variants, render modes
  3. Discovery Survey — multi-step questionnaire for color preferences & emotions
  4. Personal Color Profile — personalized results, emotional palette, recommendations
  5. About / 13 Phases — interactive roadmap of the full project vision
- Device strategy: adaptive
- Design style: dark cosmic — deep space BG, luminous glowing swatches, gold accents
- Technical constraints: Next.js App Router, Tailwind CSS, shadcn/ui, no external APIs
- Nova Agent: not needed
- Completed: brief.md, design.md
- Current iteration: Full platform MVP
